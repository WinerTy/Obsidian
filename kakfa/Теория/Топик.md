**Topic**(Топик) - это категория/канал для группировки потоков сообщений. Например: Работающий сервис выполняет задачу обновление пользователя, но помимо этого сервиса существуют другие. Для их работы требуется актуальный объект пользователя, чтобы их синхронизировать моно использовать топики. Любое действие с пользователем отправляется в этот топик, будь то обновление/удаление/создание объекта. 
Пример Payload в топик:
```json
{
	"type": "user.update",
	"payload": {
		"id": 1,
		"name": "John Doe",
		"role": "common",
		...
	}
}
```
По данному JSON можно будет определить какой тип события произошел и выполнять какие то действия.
