## Добавление записи

В **Pandas** 2.0.0 метод **append** был удален, и теперь вместо него используем **concat** 
```python
table = pd.concat([table, new_row])
```
Где:

| Аргумент | Описание         | Тип           |
| -------- | ---------------- | ------------- |
| table    | Исходная таблица | [[DataFrame]] |
| new_row  | Новая строка     | [[Series]]    |
